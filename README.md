# 越鑫证书转换合并工具

![image](https://github.com/user-attachments/assets/ea91374d-5f02-47e9-abad-76c107409ce8)


## 🌟 功能特性
- **双功能合一**：集成Word转PDF和PDF合并两大核心功能
- **批量处理**：支持文件夹级批量转换/合并操作
- **进度可视化**：实时显示处理进度和当前文件
- **智能提示**：错误处理机制和操作状态提示

## 🛠️ 安装指南

### 前置要求
- Python 3.6+
- Microsoft Word（用于DOCX转换）
